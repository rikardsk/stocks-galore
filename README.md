# Stocks Galore - Personal Stock Workbench

A modern, glassmorphic stock management workbench built with React, TypeScript, and Vite. Organize your investment ideas with powerful drag-and-drop lists and categories.

## Features

- **Country-Specific Lists**: Organize stocks by country with built-in flag icons (US, Sweden, UK, Canada, and more).
- **Advanced Drag & Drop**:
  - Rearrange categories (groups) themselves.
  - Reorder lists within or between categories.
  - Drag and drop tickers to copy or move them between lists.
- **Premium Aesthetics**: Sleek dark mode with glassmorphism effects and custom-styled UI elements.
- **Persistence**: All your layouts, categories, and ticker lists are saved automatically to local storage.
- **Interactive Tickers**: View real-time stats including price changes, volume, and market cap.

## Quick Start

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run development server**:
    ```bash
    npm run dev
    ```

## Technology Stack

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Vanilla CSS with HSL-tailored colors
- **Icons**: Lucide React
- **Logic**: Custom storage hooks and native HTML5 Drag and Drop API
