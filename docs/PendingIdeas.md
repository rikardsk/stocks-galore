1. The "Briefcase" Badge (Simple & Visual)
We can add a small Briefcase 💼 icon next to the ticker symbol. When clicked, it toggles the "Owned" status.

Visual: Owned stocks could have a subtle gold or emerald glow around their row to make them stand out from watchlisted items.
Aesthetic: It keeps the UI clean but provides immediate visual feedback.
2. Portfolio Stats (Functional)
If you want to go a step further, we could allow you to enter:

Quantity: How many shares you own.
Cost Basis: What price you bought them at.
Role: The app can then automatically calculate your Total Value and Current P/L (Profit/Loss) in real-time.
3. Dedicated "Portfolio" Filter
In the Market Overview Table and the Global Filters, we can add a toggle to "Show Owned Only." This would allow you to quickly see how all your holdings are performing relative to each other.

4. Special List Panel
We could create a permanent "Portfolio" List (similar to your "Watchlist") that has a unique header style (e.g., a gradient background) to distinguish it from regular watchlists.
